How to get a cloud server for free

Stufz | A Blog Made By StuffzEZ

Hi, I'm Stuf_y and this is my blog. Today I’m going to talk about servers.

They’re everywhere—they host websites, apps, databases, game servers… basically anything online. So if you’re a developer (like me), at some point you’ll need one.

When I needed a server for my projects, I thought:

“Is there a way to get your own server for free?”

So I did some research. A lot of research.

Eventually, I came across Oracle Cloud.

Oracle Cloud Free Tier

Oracle Cloud offers a free tier that’s surprisingly powerful. It includes things like:

  • Up to 24–27GB RAM (depending on configuration)
  • 200GB storage
  • ARM-based compute instances (which are actually pretty fast)
  • A couple of smaller AMD-based VMs

At first, it sounds almost too good to be true:

“What’s the catch?”

Well… there isn’t a big obvious one—but there are a few important details:

The “Catch” (Kind of)

  • You need to log in occasionally (accounts can be reclaimed if inactive for long periods)
  • Free resources are limited in availability (sometimes you’ll see “out of capacity” errors)
  • Setup is more complex than beginner-friendly platforms
  • You need to provide card details for verification (you won’t be charged if you stay in free limits)

So yeah—it’s not completely effortless, but it’s still one of the best free options out there.

How to Set It Up

Getting started is actually pretty straightforward:

  1. Go to the official signup page:
    https://oracle.com/cloud/free

  2. Create an account

    • Enter your details
    • Verify your email and phone number
    • Add a payment method (again, this is just verification)

  3. Once you're in:

    • Open the Oracle Cloud Dashboard
    • Go to Compute → Instances
    • Click Create Instance

  4. Choose your instance:

    • For best free performance, select an Ampere (ARM) instance
    • Adjust CPU and RAM (you can scale within the free limits)

  5. Add an SSH key (important!) - Make sure to save it!

    • This lets you securely connect to your server

  6. Launch the instance and connect:

    • Use SSH (e.g. ssh ubuntu@your-server-ip)

Final Thoughts

If you’re willing to spend a bit of time learning the setup, Oracle Cloud’s free tier is honestly kind of insane for what it offers. It’s powerful enough to run:

  • Websites
  • APIs
  • Discord bots
  • Game servers
  • Personal projects
    AND much more, including some things in previous blog posts I have made!

Just don’t expect it to be “set and forget”—you’ll need to manage it a bit.
