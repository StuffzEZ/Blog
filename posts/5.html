How to make a Minecraft server (BONUS: Cross Platform aswell)- 100% Free

Stufz | A Blog Made By StuffzEZ

Hi, I'm StuffzEZ and this guide will tell you how you can host a fun, cross platform Minecraft server, for free! This guide has 2 parts: The Aternos Method and the DIY method. Today I will be talking about the DIY method.

Requirements

  • You must have a server. I recommend getting an oracle cloud always free server with the maximum specs you can get. There are plenty of tutorials on YouTube for this but make sure you install ubuntu server. You can also use a home server/home lab for this, again use ubuntu server
  • You must be willing to maintain it
  • You must have a computer that you can use to connect to the cloud instance and to configure it on
  • You must have basic knowledge of how to use things like this
  • A terminal program (I recommend MobaXterm Personal- the portable, free version though)

Step 1: Installing CasaOS

To make this guide easier, we will install something called CasaOS which will let you connect to your cloud instance via a GUI (graphical user interface) instead of a CLI (command line interface). To do this, paste the following command into your CLI:

curl -fsSL https://get.casaos.io | sudo bash

Now, casaos should begin installing and you can go and have a break. This may take a while. After this, if you come back and see NO major errors (lots of red text commonly means a major error) then first make sure that the port 80 is exposed on your cloud provider (on oracle, just expose all ports and type sudo ufw enable and sudo ufw allow 80 into the terminal so that you won't have to do this much). Then go to your server's IP. Eg:

http://123.456.789.101

This should then bring you to the CasaOS user creation screen. Create a user to access the dashboard.

Step 2: Installing Crafty

Now, there are many ways to host a Minecraft server. But one of the ways is with Crafty. It is a self hosted Minecraft Server control and management app. We recommend this because it is easy to use and setup. To install it, go to the CasaOS app store and search for crafty. Then install it. If you can't find crafty, install the BigBear CasaOS app store (which should be installed by default). Once you have installed Crafty, go back to your CLI and type:

sudo ufw allow 8443/tcp && sudo ufw allow 8123/tcp && sudo ufw allow 19132/udp && sudo ufw allow 25500:25600/tcp && sudo ufw reload

Then go to your ip port 8111. Eg:

http://123.456.789.101:8111

And then create an account. Now you should be in Crafty.

Step 3: Creating A Server

On the Crafty sidebar, press the servers icon and press create server. Even if you play on bedrock (we will configure later, although java is recommended) select:

Server Type: Minecraft Servers
Server Select: Paper
Server Version: Latest Version Supported By Bedrock And Java (Currently 1.21.8)
Server Name: (Put your server name here)
Minimum Memory: (Recommended 3GB)
Maximum Memory: (Recommended 10GB- you can get upto 32GB free ram when creating your oracle cloud instance - OCI - I think)
Server Port: 25565

Then Press 'Build Server'. Now we can boot the server up then get some plugins.

Step 4: First Boot

To boot your server, press the Start button on the server control pannel. This should boot it up. Once you see a blue message saying something like:

Done (4.548s)! For help, type "help"

Then it is fully booted. This means your world should also be fully generated.

Step 5: Adding Plugins

Since we are running Paper, a (technically) modded version of Minecraft which can be connected to with any client (Fun Fact: Paper is used on servers like Hypixel etc.) then you have the ability to add plugins. Here below will be a guide on where to get plugins and what plugins to get.

Where to get plugins

You can get plugins from many places but the most trusted place is Modrinth, after that is SpigotMC. Remember: If it doesn't have a high download count, it MIGHT not be safe and MAY have a virus.

What plugins to get

There is a wide variety of plugins for everything but here are our top picks:

  • Always, on every server, get [Luckperms], [PAPI] and [EssentialsX]
  • For Guns, get [Quality Armory]
  • For bedrock compatibility, use [Floodgate], [GeyserMC], GeyserConnect, ThirdPartyCosmetics and GeyserOptionalPack
  • For a RPG fun skills levelling plugin, use [AuraSkills]
  • For optimisation, use [Chunky] and [Spark]
  • For extensive logging and Anti-Griefing, use [CoreProtect]
  • For custom crafting recipes, use [CustomCrafting]
  • For controlling access to the end dimension, use EndDimensionControl
  • For EssentialsX Chat formatting, use EssentialsX Chat
  • For spawn control, use [EssentialsX Spawn]
  • For converting images to maps, use [ImageFrame]
  • For seeing inventories eg. if a player hacks an item and you want to remove it, use [InvSee++]
  • For Lifesteal, use [LifeStealz]
  • For sharing logs, use MCLogs
  • For minecart speed control, use Minecart Speed Plus
  • For Anti-Xray, use MinerTrack
  • For moderation, use ModereX
  • For multiple worlds, use Multiverse Core, Multiverse Inventories, Multiverse Nether Portals and Multiverse Portals
  • For maintenance mode, use Maintenance
  • For customizing your server list, use AdvancedServerList
  • For toggling PvP, use PVPToggle
  • For sitting on slabs and stairs, use Sit
  • For tweaks to your SMP, use SMPTweaks
  • For automatic vein mining, use VeinMiner
  • For automatic version switching, use [ViaVersion]
  • For admin building tools, use [WorldEdit]

Things you should know

  • You can commonly install old plugins on new versions
  • You can install Bukkit and Spigot plugins on Paper but not Paper plugins on spigot or bukkit
  • You will need to configure plugins by editing the config files. This is the same with EVERY minecraft host. You also will need to edit your server.properties to change things like resource packs.
  • Every new bedrock update, you have to update Geyser

How to add plugins

To add plugins, go to CasaOS > Files > AppData > crafty > servers > [YOUR SERVER ID - You can find in server configuration tab] > plugins > Add Files > Upload Files and upload the plugins then reboot your server

Step 6: Joining your server

To join your server go to YOUR-IP:25565 as the Java Address or YOUR-IP:19132 as the BEDROCK address then if you type

op YOUR-USERNAME

In the server console on Crafty, you should have OP.
